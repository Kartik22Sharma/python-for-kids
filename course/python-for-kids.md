# 🐍 Python for Kids (Ages 5–10)

A fun, easy Python course for young learners! Built with simple English, lots of emojis, and real-life examples like pizza 🍕, pets 🐶, and school bags 🎒.

## 📚 What's Inside

| Folder | What it is |
|---|---|
| 📄 `course/python-for-kids.md` | The full course book (becomes a PDF) |
| 🎨 `slides/python-for-kids-slides.md` | Classroom slides (becomes a PPT) |
| 🐍 `examples/` | Little Python programs kids can run |

## 🎁 How to Get the PDF and PPT (for Teachers & Parents)

You don't need to install anything! Every time the course is updated, GitHub builds the PDF and PPT automatically.

1. Click the **Actions** tab at the top of this repo
2. Click the **latest successful run** (green ✅)
3. Scroll down to **Artifacts**
4. Click **`python-for-kids-course`** to download a zip
5. Unzip it — you'll find:
   - 📄 `python-for-kids.pdf` (the full book)
   - 🎨 `python-for-kids-slides.pptx` (open in PowerPoint)
   - 📑 `python-for-kids-slides.pdf` (slides as PDF — bonus!)

## ✏️ How to Edit the Course

1. Open `course/python-for-kids.md` (for the book) or `slides/python-for-kids-slides.md` (for slides)
2. Click the ✏️ pencil icon to edit
3. Make your changes and commit
4. Wait ~5 minutes — your new PDF/PPT will be ready in the Actions tab! 🎉

## 🧑‍🏫 For Kids — Run the Example Code

Open https://trinket.io or https://replit.com (no install needed!) and paste any file from `examples/`. Click **Run** and see the magic! ✨
